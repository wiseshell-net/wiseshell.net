Since this game is made with C++ and SDL2, it supports a large variety of platforms, such as Windows, MacOSX, Linux, iOS and Android.

Requirements for running this game are very low. You have to make sure that your device has a GPU for accelerated graphics.

If, for any reason, your device does not have a GPU, you can try with software rendering, but this is unsupported.
